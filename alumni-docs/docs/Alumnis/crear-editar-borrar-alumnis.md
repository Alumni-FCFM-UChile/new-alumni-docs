# Crear, Editar y Borrar Alumnis

## Crear Alumnis

Para la creación de nuevos Alumnis, se debe ir a la sección de "Alumnis" en la barra de navegación de la aplicación web. Luego, hacer click en el botón con un símbolo de "+" en la esquina inferior derecha de la pantalla. Esto redirigirá al usuario a un formulario donde podrá ingresar los datos del nuevo Alumni. Una vez completado el formulario, se debe hacer click en el botón "Guardar" para crear el nuevo Alumni en la base de datos.

## Editar Alumnis

Para la edición de un Alumni primero debe seleccionarse desde la lista de Alumnis. Una vez dentro del perfil del Alumni, el formulario es idéntico al de creación, pero con los campos ya llenados con la información existente. El usuario puede modificar los campos que desee y luego hacer click en el botón "Guardar" para actualizar la información del Alumni en la base de datos.

## Borrar Alumnis

Para borrar un Alumni, primero debe seleccionarse desde la lista de Alumnis. Una vez dentro del perfil del Alumni, se debe hacer click en el botón "Eliminar" ubicado en la parte inferior del formulario. Aparecerá una ventana emergente solicitando confirmación para eliminar el Alumni. Si el usuario confirma, el Alumni será eliminado de la base de datos.
